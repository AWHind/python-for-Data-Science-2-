                            TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: Port 3000 Already Used
SOLUTION: npm run dev -- --port 3001

PROBLEM: Module Not Found
SOLUTION: rm -rf node_modules && npm install

PROBLEM: Build Fails
SOLUTION: npm run lint && npm run build

PROBLEM: API Connection Error
SOLUTION: Start backend on port 8000 (see section above)

PROBLEM: Hot reload not working
SOLUTION: npm run dev -- --no-cache
