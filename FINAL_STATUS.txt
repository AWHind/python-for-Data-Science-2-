================================================================================
                            ARCHITECTURE
================================================================================

REQUEST FLOW:
  Frontend (prediction-form.tsx)
       ↓
  POST /predict {model: "XGBoost", features: [...]}
       ↓
  Backend (main.py)
       ↓
  model_cache.load_model("XGBoost")
       ↓
  Try MLflow → Try Local Cache → Use Dummy Model
       ↓
  df = DataFrame(features)
  prediction = model.predict(df)
       ↓
  Return PredictionResponse
       ↓
  Frontend displays result

MODEL LOADING PRIORITY:
  1. In-Memory Cache (fastest)
  2. MLflow (if available)
  3. Local Pickle Cache (fallback)
  4. Dummy Model (emergency)

This ensures the API ALWAYS works!
